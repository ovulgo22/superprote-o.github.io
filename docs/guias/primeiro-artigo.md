# Guias: Primeiro Artigo

Este artigo demonstra os elementos básicos que você pode usar em seu Markdown.

## Listas

Você pode criar listas não ordenadas:

- Item 1
- Item 2
  - Sub-item 2.1
  - Sub-item 2.2

Ou listas ordenadas:

1. Primeiro passo
2. Segundo passo
3. Terceiro passo
